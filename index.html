Craftcoin

A CryptoCurrency specifically for games that require you to mine and craft.

View on GitHub Download .zip Download .tar.gz

Welcome to CraftCoin.

This is a fork of Bitcoin, and is designed for use in games that allow you to mine and craft. I've even heard that there is a game that uses those two specific words in the title.

More information will come as we build.
